# weather-app

# About the Project
This project is a Weather Application. It will gather the search results from the city that you input and give you the current forecast as well as a 5 day forecast.

# Built With
This project was built using HTML and CSS and JavaScript. Also using JQuery, Bootstrap, Moment.JS languages.

# Getting Started
To access this code you will need to have Visual Studio Code downloaded onto your device.

# Usage
Can be used as an application to predict future weather or as a starter code for anyone wanting to build a more in depth application.

# credits 
The weather api was used to gather the data. You will need to create your own account to gain a key.
